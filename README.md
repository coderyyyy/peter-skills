# peter-skills

OpenCode 技能集合，为软件开发提供专业辅助。

## 技能列表

| 技能 | 说明 |
|------|------|
| **code-review** | 代码审查助手，帮助审查代码质量、发现潜在问题、提供改进建议 |
| **database-design** | 数据库设计助手，帮助设计数据库结构、优化查询、提供SQL建议 |
| **debug-code** | 代码调试助手，帮助诊断和解决程序错误、异常和运行时问题 |
| **frontend-helper** | 前端智能助手，帮助解决 JavaScript、TypeScript、React、Vue、Node 等前端代码问题 |
| **generate-docs** | 文档生成助手，帮助创建代码文档、API文档、项目文档 |
| **git-branch** | Git 分支助手，帮助创建符合约定的本地开发分支（含命名约定、基础分支、git 命令） |
| **git-commit** | Git 提交助手，根据 Conventional Commits 规范生成专业的 Git 提交信息 |
| **implement-feature** | 功能实现助手，帮助设计和实现新功能，提供架构建议和代码实现 |
| **performance-optimize** | 性能优化助手，帮助分析和优化代码性能，提供优化策略和实践建议 |
| **refactor-code** | 代码重构助手，帮助改进代码结构、提高可读性和可维护性 |
| **security-audit** | 安全审计助手，帮助发现代码中的安全漏洞和风险，提供修复建议 |
| **system-design** | 系统设计助手，帮助设计高可用、高性能的分布式系统架构 |
| **write-tests** | 测试编写助手，帮助创建单元测试、集成测试、端到端测试，提供测试策略和最佳实践 |

## 使用方法

在 OpenCode 中使用技能：
```
/skill_name [任务描述]
```
